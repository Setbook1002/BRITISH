# Support Hub Local Setup

This project now includes a working front end and a simple local back end.

## Run locally

1. Open a terminal in this folder.
2. Run:
   - Windows: `py -m http.server 3000`
3. Open: http://localhost:3000

The site is served as a static front end, and the forms submit to the browser-side logic for demo purposes. To add a real production back end later, connect these forms to a real API or database.
